Este projeto aplica a 1ª Heurística de Nielsen: Visibilidade do Status do Sistema.
Durante a execução, o programa mostra mensagens de progresso como “Verificando CPU...” e “Lendo Memória RAM...”, mantendo o usuário informado sobre o que está acontecendo.
Com isso, o sistema transmite confiança e evita que o usuário pense que o programa travou, demonstrando na prática como o feedback contínuo melhora a experiência de uso.
